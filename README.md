# ⚡ Reaction Tester Game

The Reaction Tester Game is a simple and fun web-based game that measures how quickly a user can react to a visual change on the screen. Built using HTML and CSS, this project focuses on UI design and user interaction.

---

## 📌 Features

- Random shape appears on screen  
- Measures user reaction time  
- Displays reaction time instantly  
- Simple and clean user interface  
- Responsive design  

---

## 🛠️ Technologies Used

- HTML5 – Structure of the game  
- CSS3 – Styling and animations  

---

## 📂 Project Structure

Reaction-Tester/
│
├── index.html        # Main game UI  
├── style.css         # Styling and animations

## 🚀 How to Play

1. Open the game in your browser  
2. Wait for the shape to appear on screen  
3. Click on the shape as quickly as possible  
4. Your reaction time will be displayed  
5. Try again to improve your speed  

---

## ⚙️ How to Run

1. Clone the repository  
   git clone https://github.com/your-username/reaction-tester.git  

2. Open the project folder  

3. Run index.html in your browser  

---

## 📊 Game Logic

- A shape appears after a random delay  
- The timer starts when the shape appears  
- The timer stops when the user clicks the shape  
- Reaction time is calculated and displayed
